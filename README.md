## 使用uglifyjs 将对应的js 进行压缩

### install dependence
```
	npm install uglify-js
```

### Command line usage
```
    node compress.js input-file-path [out file path]
	
```

### Or command line usage
```
	npm run start  input-file-path
```

### example
```
	npm run start test.js
	## or
	node compress.js test.js
```
